# Reports Folder

Use this folder for output documents such as QA runs, release readiness checks, and incident or regression investigations.

Naming suggestion:
- `YYYY-MM-DD-topic.md`

Examples:
- `2026-03-17-qa-service-flow.md`
- `2026-03-17-release-readiness.md`
- `2026-03-17-issue-81-regression-check.md`
